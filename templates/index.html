Notación para ingresar las funciones

Esta calculadora utiliza la misma notación que utiliza Python. Por lo que, al ingresar una función, se debe seguir la sintaxis del lenguaje de programación.

Además, la calculadora permite la entrada de funciones con las siguientes caracteristicas:

Funciones trigonométricas, se permite el uso del seno, coseno y tangente.

Para seno: "sin(x)"
Para coseno: "cos(x)"
Para tangente: "tan(x)"

Funciones exponenciales y con raiz cuadrada.

Para raiz cuadrada: "sqrt(x)"
Para exponenciales: "exp(x)"

Ejemplos

x^10 - 1 ----> x**10 - 1

0.2 + 25x - 200x^2 + 675x^3 – 900x^4 + 400x^5 ----> 0.2 + (25*x) - (200*x**2) + (675*x**3) - (900*x**4) + (400*x**5)

e^-x - x ----> exp(-x) - x

sin(√x) - x ----> sin(sqrt(x)) - x
